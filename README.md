Astro portfolio scaffold

This repository contains a minimal Astro + React + Tailwind scaffold.

Authoring quickstart:
- Add a writeup: create content/writeups/YYYY-MM-DD-slug.md with frontmatter (title, date, tags, summary).
- Add a project: create content/projects/slug.md with frontmatter (title, date, summary, repo).

Local dev:
1. npm install
2. npm run dev

Deployment:
- Push to main branch. GitHub Actions will build and deploy to gh-pages branch.
